# Fire Alert Sound Placeholder

This directory should contain a fire alert notification sound file named `fire-alert.mp3`.

## How to add the sound:

1. **Option 1: Use a free alarm sound**
   - Visit: https://freesound.org/ or https://mixkit.co/free-sound-effects/alarm/
   - Search for "fire alarm" or "emergency alert"
   - Download an MP3 file
   - Rename it to `fire-alert.mp3`
   - Place it in this directory

2. **Option 2: Use a system sound**
   - Copy any alert sound from your system
   - Convert to MP3 if needed
   - Name it `fire-alert.mp3`

3. **Option 3: Use text-to-speech**
   - Use online TTS: https://ttsmaker.com/
   - Generate: "Emergency! Wildfire confirmed by satellite. Immediate response required."
   - Download as MP3
   - Name it `fire-alert.mp3`

## Recommended sound properties:
- Duration: 2-5 seconds
- Volume: Moderate (not too loud to be jarring)
- Type: Clear alert tone or voice notification
- Format: MP3

## Fallback:
If no sound file is present, the alert modal will still work - it just won't play audio.
